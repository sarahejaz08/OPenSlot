# OpenSlot

OpenSlot is a simple Python desktop application designed to manage and book shared resources.  
Admins can add resources, attach notes to specific dates, and users can view and book available slots.  
The app uses **Tkinter** for the interface and **JSON** for data storage.

## Features

- Add, edit, and remove resources.
- View a calendar of booked and available slots.
- Add admin notes to specific dates.
- Store all data in a JSON file (no database required).
- Lightweight, offline, and beginner-friendly.

## Requirements

- Python 3.x  
- Tkinter (included with most Python installations)
